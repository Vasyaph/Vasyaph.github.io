Шейдеры

Данный проект полностью реализован на юнити и использует только базовые возможности движка.
В данном проекте есть 3 шейдера которые могут украсить пару локаций или ключевых объектов. Они подходят только для 2D пространства и не могут корректно использоваться в 3D условиях.


  • Шейдер огня- шейдер с красивой графикой и возможностью менять уровень пикселизации
  • Шейдер воды- есть возможность усиление ряби и изменения цвета
  • Игра жизнь- стандартная игра жизнь, которая сначала принимает изображение и работает с закрашеними пикселями объекта
